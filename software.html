SoundBlocks Configurator

SoundBlocks Configurator es una interfaz gráfica de configuración para los módulos sonoros interactivos. Permite determinar las acciones sonoras realizadas por cada módulo y la designación de los datos sensados y transmitidos entre módulos.

Es compatible con los entornos Windows, Mac OS X y Linux.

Autores: Laurence Bender, Sabrina García, Germán Ito

GIT: https://github.com/soundblocks/configurator

Uso básico del módulo:


  122->ax:102,106-108 #bloque 122 manda ax a los bloques 102, 106, 107, 108
  107->t1,t3:102,105-107
  108->az,t3,t7,azimuth:102-106

  108<-ax:122@dfVolume[0,15] #bloque 108 recibe ax del bloque 122 y lo usa para manejar volumen
  102<-t3:107@dfPlay[1,5]
  103<-az:108@dfVolume[0,15]
  103<-t7:108@dfPlay[1,5]
